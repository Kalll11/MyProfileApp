# Notes App

## 📌 Deskripsi
Aplikasi sederhana berbasis Jetpack Compose yang dibuat untuk memenuhi tugas praktikum Pertemuan 3 mata kuliah Pengembangan Aplikasi Mobile.

Aplikasi ini menampilkan informasi profil pengguna serta implementasi beberapa komponen UI dasar seperti Card, Text, Button, dan Image menggunakan paradigma deklaratif.

---

## 🎯 Fitur
- Profile Card (Latihan 1)
- Login Form (Latihan 2)
- Product List (Latihan 3)
- Profile Screen (Tugas Praktikum)

---

## 🧩 Teknologi
- Kotlin
- Jetpack Compose
- Coil (AsyncImage)

---

## 📂 Struktur Composable
- ProfileCard
- LoginForm
- ProductList
- ProfileHeader
- InfoItem
- ProfileScreen
